### 🛒 **NxtTrendz** - Full Stack E-Commerce Application

A production-ready full-stack e-commerce platform featuring JWT authentication, Razorpay payment integration with HMAC SHA256 verification, prime subscription system, persistent MongoDB cart with optimistic UI updates and debounced sync, and deployed on AWS EC2 Ubuntu with Nginx reverse proxy and PM2 process management.

**Tech Stack:** React, Vite, Node.js, Express.js, MongoDB Atlas, JWT, Razorpay, AWS EC2, Nginx, PM2

**Key Features:**
- JWT-based authentication and authorization
- Razorpay payment integration with HMAC SHA256 verification
- Prime subscription system
- Persistent MongoDB cart with optimistic UI updates
- Debounced cart synchronization
- AWS EC2 deployment with Nginx reverse proxy
- PM2 process management for backend

**Deployment:**
- **Frontend:** AWS EC2 + Nginx
- **Backend:** AWS EC2 + PM2 + Express
- **Database:** MongoDB Atlas
- **Live Demo:** http://13.207.106.94
- **Demo Video:** https://youtu.be/AL5l0mAiP3Y

**Repository:** [github.com/avanishtatat/NxtTrendz](https://github.com/avanishtatat/NxtTrendz)

### 💻 **CodeBuddy** - AI-Powered Coding Assistant ✅ Live

A full-stack AI coding assistant built for beginner and intermediate developers. Features secure authentication, conversation memory, persistent chat history, and syntax-highlighted markdown rendering.

**Key Features:**
- Secure user authentication with JWT and Bcrypt
- AI-powered responses using Groq API (llama-3.3-70b-versatile)
- Conversation memory — last 10 messages sent as context
- Persistent chat history saved to MongoDB with pagination
- Daily message limit of 20 messages per user
- Question-answer pairs with expandable answers
- Markdown rendering with syntax highlighted code blocks
- Responsive design for mobile and desktop
- First-time user welcome card with suggestion chips

**Technologies:** React.js, Vite, Node.js, Express.js, MongoDB, Tailwind CSS, Groq SDK, JWT, Bcrypt

**Deployment:**
- 🌐 **Frontend:** [Vercel](https://avanish-codebuddy.vercel.app)
